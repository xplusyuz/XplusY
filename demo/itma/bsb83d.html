8-sinf Algebra fanidan 3 - BSB savollari

1-savol (4 ball)

Agar \(a > 0\) bo'lsa, u holda

\[\frac{a + 9}{2} + \frac{a + 16}{2} \geq 7\sqrt{a}\]

tengsizlikni isbotlang.

Yechim:

Avval ifodani soddalashtiramiz:

\[\frac{a + 9}{2} + \frac{a + 16}{2} = \frac{2a + 25}{2} = a + \frac{25}{2}\]

Endi, \(a + \frac{25}{2} \geq 7\sqrt{a}\) tengsizlikni isbotlaymiz.

Ikki tomonni ham 2 ga ko'paytiramiz:

\[2a + 25 \geq 14\sqrt{a}\]

\(\sqrt{a} = t\) deb belgilaymiz (\(t > 0\)), u holda \(a = t^2\)

\[2t^2 + 25 \geq 14t\] \[2t^2 - 14t + 25 \geq 0\]

Diskriminantni hisoblaymiz:

\[D = (-14)^2 - 4 \cdot 2 \cdot 25 = 196 - 200 = -4 < 0\]

Diskriminant manfiy bo'lgani uchun kvadrat tenglama haqiqiy ildizga ega emas va \(2t^2 - 14t + 25\) ifoda har doim musbat.

Shuning uchun tengsizlik barcha \(a > 0\) lar uchun o'rinli.

2-savol (5 ball)

\(2x^{3} - 3x^{2} - 4x + 6 = 0\) tenglama ildizlari \(m,\ n,\ p\) bo'lsa, \((m + 1)(n + 1)(p + 1)\) ning qiymatini toping.

Yechim:

Kub tenglama uchun Vieta formulalariga ko'ra:

\[m + n + p = \frac{3}{2}\] \[mn + mp + np = -2\] \[mnp = -3\]

Endi \((m + 1)(n + 1)(p + 1)\) ifodani kengaytiramiz:

\[(m + 1)(n + 1)(p + 1) = mnp + (mn + mp + np) + (m + n + p) + 1\]

Qiymatlarni qo'yamiz:

\[-3 + (-2) + \frac{3}{2} + 1 = -4 + \frac{3}{2} = -\frac{5}{2}\]

Javob: \(-\frac{5}{2}\)

3-savol (6 ball)

\(x^{2} + x - 4 = 0\) tenglamaning ildizlari \(x_{1}\) va \(x_{2}\) bo'lsa, ildizlari \({(x_{2} - 1)}^{- 1}x_{1}\) va \({(x_{1} - 1)}^{- 1}x_{2}\) bo'lgan kvadrat tenglama tuzing.

Yechim:

Berilgan tenglama uchun Vieta formulalari:

\[x_1 + x_2 = -1\] \[x_1 x_2 = -4\]

Yangi ildizlar:

\[y_1 = \frac{x_1}{x_2 - 1} \quad \text{va} \quad y_2 = \frac{x_2}{x_1 - 1}\]

Yangi tenglama: \(y^2 - (y_1 + y_2)y + y_1 y_2 = 0\)

\(y_1 + y_2\) ni hisoblaymiz:

\[y_1 + y_2 = \frac{x_1}{x_2 - 1} + \frac{x_2}{x_1 - 1} = \frac{x_1(x_1 - 1) + x_2(x_2 - 1)}{(x_1 - 1)(x_2 - 1)}\] \[= \frac{x_1^2 + x_2^2 - (x_1 + x_2)}{x_1 x_2 - (x_1 + x_2) + 1}\]

\(x_1^2 + x_2^2 = (x_1 + x_2)^2 - 2x_1 x_2 = 1 + 8 = 9\)

\[y_1 + y_2 = \frac{9 - (-1)}{-4 - (-1) + 1} = \frac{10}{-2} = -5\]

\(y_1 y_2\) ni hisoblaymiz:

\[y_1 y_2 = \frac{x_1}{x_2 - 1} \cdot \frac{x_2}{x_1 - 1} = \frac{x_1 x_2}{(x_1 - 1)(x_2 - 1)} = \frac{-4}{-4 - (-1) + 1} = \frac{-4}{-2} = 2\]

Shuning uchun yangi tenglama:

\[y^2 + 5y + 2 = 0\]
4-savol (6 ball)

Tenglamalarni yeching:

a) \(y^{2} - 15y = 0\)

b) \(t^{2} - 8t - 9 = 0\)

c) \(2x^{2} - 9x - 5 = 0\)

Yechim:

a) \(y^{2} - 15y = 0\)

\[y(y - 15) = 0\] \[y = 0 \quad \text{yoki} \quad y = 15\]

b) \(t^{2} - 8t - 9 = 0\)

Diskriminant:

\[D = (-8)^2 - 4 \cdot 1 \cdot (-9) = 64 + 36 = 100\] \[t = \frac{8 \pm \sqrt{100}}{2} = \frac{8 \pm 10}{2}\] \[t_1 = \frac{18}{2} = 9, \quad t_2 = \frac{-2}{2} = -1\]

c) \(2x^{2} - 9x - 5 = 0\)

Diskriminant:

\[D = (-9)^2 - 4 \cdot 2 \cdot (-5) = 81 + 40 = 121\] \[x = \frac{9 \pm \sqrt{121}}{2 \cdot 2} = \frac{9 \pm 11}{4}\] \[x_1 = \frac{20}{4} = 5, \quad x_2 = \frac{-2}{4} = -\frac{1}{2}\]
5-savol (4 ball)

Agar \(a = 2,96\) bo'lsa, \(\sqrt{a - 2\sqrt{a - 1}} + \sqrt{a + 2\sqrt{a - 1}}\) ifodaning qiymatini toping.

Yechim:

\(\sqrt{a - 2\sqrt{a - 1}} + \sqrt{a + 2\sqrt{a - 1}}\) ifodani soddalashtiramiz:

\[\sqrt{a - 2\sqrt{a - 1}} = \sqrt{(\sqrt{a-1} - 1)^2} = |\sqrt{a-1} - 1|\] \[\sqrt{a + 2\sqrt{a - 1}} = \sqrt{(\sqrt{a-1} + 1)^2} = |\sqrt{a-1} + 1|\]

\(a = 2.96\) bo'lsa:

\[a - 1 = 1.96, \quad \sqrt{a-1} = \sqrt{1.96} = 1.4\]

Shuning uchun:

\[\sqrt{a - 2\sqrt{a - 1}} = |1.4 - 1| = 0.4\] \[\sqrt{a + 2\sqrt{a - 1}} = |1.4 + 1| = 2.4\]

Yig'indi:

\[0.4 + 2.4 = 2.8\]

Javob: 2.8
